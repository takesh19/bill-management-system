# Bill Management System

A modern laundry management web application built using MERN Stack.

## Features

- Customer Management
- Billing System
- PDF Bill Download
- WhatsApp Bill Sharing
- Revenue Dashboard
- Mobile Responsive UI
- Admin Login

## Tech Stack

- React + Vite
- Node.js
- Express.js
- MongoDB Atlas
- Recharts
- jsPDF

## Installation

### Frontend

```bash
cd vite-project
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
